# FastLMS
기본적인 학습관리시스템(Learning Management System) 구현 프로젝트

[] 회원 로그인시 로그인 히스토리(로그) 기능
    - 로그인 ID
    - 로그인 날짜
    - 접속 IP
    - 접속 UserAgent
[] 관리자 회원 상세 정보에 로그인 목록 보기 기능
[] 배너관리(백오피스 기능)
    - 배너 등록 : 이미지 파일, 대체 텍스트, URL 정보, 클릭시 Target 정보, 정렬 순서, 프론트 표시 여부
    - 배너 수정
    - 배너 삭제
[] 프론트 배너 노출 기능
    - 공개로 설정된 배너를 순서대로 표시
